# Cognitive Engine MCP

## Build
`npm run build && npm start`

## Tools
- **Memory**: `remember`, `recall`, `forget`, `clear_memories`
- **Think**: `think` (depth 1-5), `verify`
- **Learn**: `learn`, `get_insights`, `forget_lesson`, `clear_lessons`
- **DB**: `vacuum_db` (압축)

## Install
`npm install -g .`
